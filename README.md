📚 Estudos de JavaScript - Curso em Vídeo
Este repositório é um registro completo da minha jornada de aprendizado em JavaScript, contendo desde os fundamentos básicos até a manipulação de dados complexos com Objetos. O projeto reúne códigos desenvolvidos em aula, desafios práticos e um guia teórico detalhado.

📖 Resumo do Aprendizado
O conteúdo desenvolvido abrange a exploração da árvore hierárquica do navegador através do DOM (Document Object Model), utilizando métodos de seleção como getElementById e querySelector para gerenciar interações dinâmicas via Eventos.

A base lógica foi consolidada com Estruturas de Controle, incluindo condicionais (if, else, switch) e laços de Repetição fundamentais como while, do while e o laço for com variável de controle.

A manipulação de dados evoluiu para o uso de Variáveis Compostas (Arrays), onde explorei métodos como .push(), .length e a lógica de ordenação correta para números e strings. Por fim, o repositório cobre a criação de Funções modulares e a introdução a Objetos, organizando propriedades e métodos para representar entidades de forma programática.

📂 Estrutura de Pastas
A organização do repositório segue a sequência das aulas e módulos do curso:

/aula04 a /aula17: Exemplos práticos codificados durante as videoaulas, incluindo manipulação de variáveis, DOM e objetos.

/aula12ex, /aula14ex, /aula16ex: Pastas dedicadas a projetos de fixação e desafios (como o Verificador de Idade e Analisador de Números).

Estudos_Javascript.docx: Documento teórico consolidado com explicações e revisões de cada conceito.
